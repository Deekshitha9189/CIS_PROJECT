# Multi Cloud Simulation Dashboard

## Description
This project is a Multi-Cloud Simulation Dashboard that demonstrates how different cloud services can be monitored and managed through a single interface.

## Features
- Centralized dashboard
- Simulated cloud services
- Interactive UI
- Responsive design
